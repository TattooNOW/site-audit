# GEO Visibility Checklist

TattooNOW client-facing GEO audit execution tracker. Syncs progress to Google Sheets via Apps Script.

**Live:** Enable GitHub Pages (Settings → Pages → Branch: main) to view at `https://tattoonow.github.io/geo-checklist/`

## Usage
- Paste audit JSON or load sample data
- Check off completed items
- Progress syncs to Google Sheets automatically
- Embed in GHL via iframe with `?locationId=XXX`
